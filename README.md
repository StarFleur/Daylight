# Daylight

## About
Daylight is an app that helps you organise and plan your day, inspired by the [A Better Day](https://www.youtube.com/watch?v=CVAK8MhuqrY) app.

## Features
- View all your tasks in a daily overview
- Add, edit and complete tasks efficiently
- Set daily reminders to stay on track

## System Requirements
Xcode 26.0+

## Installation
1. Clone the repository
```
git clone https://github.com/StarFleur/Daylight.git
```
2. Open the project in Xcode
```
cd Daylight
open Daylight.xcodeproj
```
3. Build and run the project in Xcode (Command + R)
